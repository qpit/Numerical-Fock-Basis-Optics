
############################  DESCRIPTION  ####################################
Scripts used in the paper https://doi.org/10.48550/arXiv.2211.16343
###############################  SCRIPTS  #####################################
 - QuantumOpticsV7.py
	Package for doing calculations. A quantum state is defined as an instance of a class. The instance has a number of attributes (such as a density matrix) and a number of functions.

 - Registersv4.py
	Script for doing calculations on a single repeater segment.

 - Repeater1-v6.py
	Script for simulating a chain of repeater segments (a full repeater).
