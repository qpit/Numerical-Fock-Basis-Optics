
############################  DESCRIPTION  ####################################
Toolbox for doing quantum optics in the Fock basis.
###############################  SCRIPTS  #####################################
 - QuantumOpticsV7.py
	Package for doing calculations. A quantum state is defined as a class with a number of attributes (such as a density matrix) and a number of functions.

 - Registersv4.py
	Script for doing calculations on a single repeater segment.

 - Repeater1-v6.py
	Script for simulating a chain of repeater segments (a full repeater).
